#  Pancake Token (PNC)

A simple **BEP20 token** built with **Solidity** and **OpenZeppelin**, deployable on **Binance Smart Chain (BSC)**.  
Includes a minimal **React + Web3.js frontend** for wallet connection, balance display, and token transfers.
